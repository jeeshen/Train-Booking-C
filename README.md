# Train Booking System

A console-based train booking management system built in C that allows members to book train tickets and administrators to manage train schedules, bookings, and user accounts.

## Features
- Member registration and login system
- Train ticket booking with multiple ticket types and payment methods
- Train schedule management (add, display, modify)
- Booking management and cancellation
- Member points system
- Staff and admin account management
- Booking records and analytics
- Member and staff account modifications

## Installation
```bash
# clone the repo
git clone https://github.com/yourusername/train-booking-system.git

# open the solution file in Visual Studio
cd Assignment
start Assignment.sln

# build and run the project using Visual Studio
```

## File Structure
- `main.c` - Main source code
- `booking.txt` - Booking records
- `memberPoint.txt` - Member points data
- `train_schedule.txt` - Train schedule information
